System Requirements - Technical Data

This CD is designed to be as cross platform compatible as possible.

You will need the following minimum computer set up to run this CD.

1. An IBM® compatible computer running Microsoft® Windows® 98SE (Second Edition) or later, with Microsoft® Internet Explorer version 5.x or later; or Netscape® 7.x or later. You will also need Java Virtual Machine (JVM) version 1.3.x or later browser plug-in with Java enabled on your browser and computer.

2. A MAC® running the following configurations:

MacOS X:
This CD must be run in Netscape® 7.1 (Available from: www.netscape.com ) with Apple® Java (Available from: http://www.apple.com/java )

MacOS 9.2.x:
This CD must be run in Netscape® 7.0.2 (Available from: www.netscape.com ) with Apple® MRJ 2.2.6 (Available from: http://docs.info.apple.com/article.html?artnum=120209 )

MacOS 8.x.x - 9.1.x:
This is not supported, however a work around does exist. 1) This CD must be run in Netscape® 7.0.2 (Available from: www.netscape.com ) with Apple® MRJ 2.2.6 (Available from: http://docs.info.apple.com/article.html?artnum=120209 ). 2) The search box will display double characters when typing a word to search.  This can be corrected by switching to another program and the back to Netscape® .  By doing this the double characters will stop.  3) This will need to be done every time the CD is run.  4) We recommend upgrading to MacOS 9.2.x for complete compatibility.

MAC Procedures:

1. Launch Netscape®
2. From the file menu go to "Open File (cmd + O)"
3. Open "intro.htm" from the volume "Men_CD"

3. A CD ROM drive (for Windows® auto start should be enabled).

4. This CD has also been tested in the following browsers: Mozilla® 1.4 and Opera® 7.
